Ordering

This module creates a Python object DataFrameOrdering for order by dataframe.

class order.DataFrameOrdering

Represents an ordered GreenplumPython DataFrame generated by order_by() or order_by().

property dataframe: DataFrame

Returns GreenplumPython DataFrame associated for ORDER BY

Returns

GreenplumPython DataFrame

order_by(column_name, ascending=None, nulls_first=None, operator=None)

Refine the order by adding another ordering definition to break the tie.

Parameters

  • column_name (str) – name of column to order the GreenplumPython DataFrame by

  • ascending (Optional[bool]) – Optional[Bool]: Define ascending of order, True = ASC / False = DESC

  • nulls_first (Optional[bool]) – Optional[bool]: Define if nulls will be ordered first or last, True = First / False = Last

  • operator (Optional[str]) – Optional[str]: Define order by using operator. Can’t combine with ascending.

Returns

DataFrame ordered by the given arguments

Return type

DataFrameOrdering

Example

t.order_by("id").order_by("num", ascending=False)
