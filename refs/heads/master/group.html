Grouping

This module creates a Python object TableRowGroup for group by table.

class group.TableRowGroup(table: Table, grouping_sets: List[List[Expr]])

Represents a group of rows in a Table generated by group_by().

__or__(other: TableRowGroup) → TableRowGroup

Returns the union of the two row groups.

This does not merge the two groups into one. Instead, it means applying any operation to the union is equivalent to applying the operation to each group and union the result sets.

apply(func: Callable[[Table], FunctionExpr]) → FunctionExpr

Apply a function to the row group.

Parameters

func – Callable[[Table], FunctionExpr]: a lambda function of a FunctionExpr

Returns

a callable

Return type

FunctionExpr

Example
