Grouping

This module creates a Python object DataFrameRowGroup for group by dataframe.

class group.DataFrameGroupingSets(dataframe: DataFrame, grouping_sets: List[Iterable[Expr]])

Represents a group of rows in a GreenplumPython DataFrame generated by group_by().

apply(func: Callable[[DataFrame], FunctionExpr], expand: bool = False, as_name: Optional[str] = None) → DataFrame

Apply a function to the grouping set.

Parameters

  • func – An aggregate function to be applied to

  • expand – expand field of composite returning type

  • as_name – rename returning column

Returns

resulted GreenplumPython DataFrame

Return type

DataFrame

Example

numbers.group_by("is_even").apply(lambda _: count())
assign(**new_columns: Callable[[DataFrame], Any]) → DataFrame

Assigns new columns to the current grouping sets. Existing columns cannot be reassigned.

Parameters

new_columns – a dict whose keys are column names and values are Callable`s returning column data when applied to the current :class:`DataFrameGroupingSets.

Returns

GreenplumPython DataFrame with the new columns.

Example

count = gp.aggregate_function("count")
results = numbers.group_by().assign(count=lambda t: count(t["val"]))
