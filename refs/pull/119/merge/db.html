db.database(host: str = 'localhost', dbname: Optional[str] = None, user: Optional[str] = None, password: Optional[str] = None, port: Optional[int] = None) → Database

Create a connection using psycopg2 with given arguments.

Parameters

  • host – str : default value = “localhost”

  • dbname – str : default value = None

  • user – str : Optional

  • password – str : Optional

  • port – int : Optional

There are different ways to passing database information:

db = database(host="localhost", dbname=dbname)

If it is a connection to localhost, password can be ignored.

Or, a connection can be established by passing more detailed information, in this case, password is needed for connexion:

db = database(
        host=dbIP,
        dbname=dbname,
        user=username,
        password=password,
        port=dbPort
    )
