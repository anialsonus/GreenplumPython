in_(container: Union[Expr, List[Any]]) → InExpr

Tests whether each value of current Expr is in the container.

It is analogous to the built-in in operator of Python and SQL.

Parameters

container – A collection of values. It can either be another Expr representing a transformed column of Table, or a list of values of the same type as the current Expr.
